# WR Combine & Draft Capital Analysis

## Overview
A three-part series examining whether combine athleticism
predicts draft investment at wide receiver, and what that 
means for the 2026 NFL Draft class and possible shifts in philosophy

## Series
- **Part 1** — Historical EDA: athleticism vs. draft capital (2010–2024)
- **Part 2a** — Predictive model: trained on history, validated on 2025
- **Part 2b** — 2026 draft projections
- **Part 3** — Data-driven scouting profiles for key 2026 WR prospects

## Data Sources
- NFL combine data via `nflreadr`
- Draft data via `nflreadr`
- College production via `cfbfastR`

## Key Questions
- Does 40 time actually predict where WRs get drafted?
- Which athletic profiles historically produce Day 1/2 picks?
- What does the model say about the 2026 WR class?

## Tools
R, tidyverse, ggplot2, tidymodels, nflreadr
